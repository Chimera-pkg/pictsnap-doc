# Overview


## Apa itu PictSnap

Pictsnap merupakan aplikasi desain kebutuhan Booth pameran Virtual yang terintegrasi langsung dengan SimHive yang memungkinkan pengguna dapat mengedit & mengupload item desain kebutuhan booth melalui aplikasi mobile. 

## Spesifikasi

Seperti pada aplikasi Desain mobile pada umumnya, PictSnap memerlukan spesifikasi tertentu untuk menjalankan sistem dengan baik.Pada bagian ini dijelaskan persyaratan dasar yang kami sarankan agar aplikasi dapat dijalankan secara optimal.

## Persyaratan Minimum Pada Perangkat Mobile 

    1.Sistem Operasi : Android

    2.Network : Broadband Internet connection

    3.Download Size: 37 MB

    4.Minimum Requirement:	Android 7.0 + (Nougat)
